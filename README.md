Portfolio personnel — [florianwu.github.io](https://flaironne.github.io)

**Stack :** Angular 21 · SSR/Prerender · Tailwind CSS v4 · SCSS · GitHub Pages

---

## Développement local

```bash
npm install
npm start          # http://localhost:4200
```

## Build de production

```bash
npm run build      # génère dist/portfolio/browser/ (HTML statique prérendu)
```

## Déploiement

Le déploiement est automatique via GitHub Actions à chaque push sur `main`.
